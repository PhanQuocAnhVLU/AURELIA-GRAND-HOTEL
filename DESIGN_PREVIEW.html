💻 Cách Sử Dụng

1. Animations

<div class="animate-fadeIn">Fade in element</div> <div class="hover-lift">Card nâng lên khi hover</div> <button class="animate-goldGlow">Gold glow button</button>

2. Feature Cards

<div class="feature-card"> <div class="feature-icon">🏨</div> <h3 class="feature-title">Tiêu Đề</h3> <p class="feature-desc">Mô tả</p> </div>

3. Buttons

<button class="btn-gold">Nút Vàng</button> <button class="btn-outline-gold">Nút Outline</button>

4. Dark Theme

<html data-theme="dark"> <!-- Mọi thứ tự động chuyển sang dark mode --> </html>

📁 Files Được Cập Nhật/Tạo Mới

✅ Cập Nhật

  • src/styles/variables.css - Biến CSS mới (màu, font, shadow)
  • src/styles/components.css - Buttons, cards, forms (3x lớn hơn!)
  • src/App.jsx - Import CSS files mới

✨ Tạo Mới

  • src/styles/animations.css - 10+ animations, effects
  • src/styles/layout.css - Luxury layouts, hero, CTA sections
  • DESIGN_UPGRADE.md - Hướng dẫn chi tiết

🚀 Tiếp Theo

1. Cài Đặt

Chạy npm install để cài dependencies

2. Chạy Dev Server

Chạy npm run dev để xem kết quả

3. Customize

Chỉnh sửa variables.css cho màu sắc của bạn

4. Deploy

Chạy npm run build cho production
