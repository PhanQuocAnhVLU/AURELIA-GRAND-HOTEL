## 🚀 BẮT ĐẦU NGAY

### Bước 1: Cài Đặt Dependencies
```bash
cd AureliaFinal
npm install
```

### Bước 2: Chạy Dev Server
```bash
npm run dev
```

### Bước 3: Xem Kết Quả
- Mở http://localhost:5173 trong browser
- Thưởng thức giao diện 5 sao mới! 🌟

### Bước 4: Customization (Tùy Chọn)
Chỉnh sửa các biến CSS trong `src/styles/variables.css`:
```css
:root {
  --primary-color: #D4AF37;      /* Thay đổi vàng */
  --text-main: #2C2C2C;          /* Thay đổi text color */
  /* ... và các biến khác */
}
```

## 💡 TIPS & TRICKS

### Sử Dụng Animations
```html
<!-- Fade in trên load -->
<div class="animate-fadeIn">Nội dung</div>

<!-- Hover lift effect -->
<div class="hover-lift">Card của tôi</div>

<!-- Gold glow animation -->
<button class="animate-goldGlow">Bấm tôi</button>
```

### Tạo Feature Section
```html
<section class="luxury-section">
  <div class="feature-grid">
    <div class="feature-card">
      <div class="feature-icon">🏨</div>
      <h3 class="feature-title">Tiêu Đề</h3>
      <p class="feature-desc">Mô tả của tôi</p>
    </div>
  </div>
</section>
```

### CTA Section
```html
<section class="cta-section">
  <div class="cta-content">
    <h2 class="cta-title">Đặt Phòng Ngay</h2>
    <p class="cta-subtitle">Trải nghiệm sang trọng</p>
    <div class="cta-buttons">
      <button class="btn-gold">Đặt Phòng</button>
      <button class="btn-outline-gold">Tìm Hiểu</button>
    </div>
  </div>
</section>
```

### Dark Theme
```html
<html data-theme="dark">
  <!-- Mọi thứ sẽ tự động chuyển sang chế độ tối -->
</html>
```

## 🎯 TIẾP THEO - KHUYẾN CÁO

1. **Update Homepage** - Sử dụng class mới như `.animate-fadeIn`, `.hover-lift`
2. **Apply to Cards** - Thêm `.feature-card` cho tất cả product cards
3. **Buttons** - Thay thế tất cả buttons bằng `.btn-gold` hoặc `.btn-outline-gold`
4. **Sections** - Wrap sections với `.luxury-section` hoặc `.cta-section`
5. **Tables** - Mọi table sẽ tự động nhận styling mới
6. **Test Dark Mode** - Kiểm tra `data-theme="dark"` attribute
